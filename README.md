# TIC TAC TOE GAME
## DESCRIPTION
tic tac toe is a simple two-player game developed using HTML,CSS,JAVASCRIPT.
this project demonstrates basic front-end development concepts and game logic.
## TECHNOLOGIES
-HTML
-CSS
-JAVASCRIPT
## FEATURES
-two player game 
-interactive UI
-displays winner or draw result
-restart game option
